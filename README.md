 # Magic Type

วิธีเล่น

จะมีคำตกลงมา ให้ผู้เล่นพิมพ์คำนั้นให้ถูกต้อง ถ้าคำที่ตกลงมาแตะพื้นจะเกมโอเวอร์ทันที
การตกของคำจะมีปรมาณและความเร็วเพิ่มขึ้นเหมือนคะแนนสะสมมากขึ้น

มีตัวช่วยสำหรับการเล่น 
-ลูกบอลสีเขียว(ช่วยให้คำตกลงมาช้าลง)
-ลูกบอลสีฟ้า(ช่วยเคลียร์คำที่อยู๋บนหน้าจอตอนนั้น)

มีหน้าต่างการสอนเล่นก่อนเริ่มเล่นจริง
